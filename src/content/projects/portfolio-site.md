这个项目是当前仓库的主线工作，用于搭建一个面向计算机、人工智能和网络安全方向的个人主页。

设计目标包括：

1. 在公开项目较少的情况下，先建立可信、克制、专业的首页叙事。
2. 用内容集合驱动文章与项目展示，降低后续持续写作成本。
3. 保持纯静态部署能力，便于直接托管到 GitHub Pages。
